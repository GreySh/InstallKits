## Установка и запуск

### Готовый исполняемый файл (Windows)

Если вы используете готовый `.exe`:

1. Скопируйте файл `Учет ИК_v*.*.*.exe` в удобное место
2. Файл `settings.json` должен лежать рядом (в одной папке) с `Учет ИК_v*.*.*.exe`
3. Запустите `Учет ИК_v*.*.*.exe`

## Главное окно

Главное окно содержит левую боковую панель с кнопками переключения между разделами:

| Кнопка | Раздел |
|--------|--------|
| Списание | Пакетное списание комплектов |
| Состав ИК | Управление комплектами |
| Остатки | Складской учёт носителей и коробок |
| Настройки | Путь к БД, версия |
| Операции | Журнал всех действий |

Размер и положение окна запоминаются и восстанавливаются при следующем запуске.

## Вкладка «Состав ИК»

Управление комплектами (продуктами) — создание, редактирование, удаление.

### Список комплектов

Таблица со столбцами:
- **Название** — имя комплекта
- **Доступно** — сколько комплектов можно собрать (ограничено самым дефицитным компонентом)
- **Компоненты** — состав комплекта (например, *Диск-1 x2; Коробка-1 x1*)

Сортировка — по клику на заголовок столбца.

### Добавить комплект

1. Нажмите **«Добавить комплект»**
2. Введите **Название** и **Описание**
3. Добавьте компоненты: выберите носитель/коробку и укажите количество
4. Нажмите **«Сохранить»**

### Редактировать комплект

- Выделите строку и нажмите **«Редактировать»**
- Или дважды кликните по строке

В окне редактирования можно изменить название, описание, а также добавить / удалить / изменить количество компонентов.

### Удалить комплект

Выделите строку и нажмите **«Удалить»** (красная кнопка). Подтвердите удаление.

## Вкладка «Остатки»

Складской учёт носителей (дисков) и коробок.

### Кнопки в верхней панели

| Кнопка | Действие |
|--------|----------|
| **Выгрузить остатки в Excel** | Формирует отчёт по текущим остаткам |
| **Зафиксировать остатки** | Создаёт снимок текущих остатков с сегодняшней датой |
| **Просмотр остатков** | Открывает окно просмотра зафиксированных остатков на выбранную дату |

### Подвкладки: Носители и Коробки

Каждая подвкладка содержит:

- **Таблица**: Название, Описание, Количество (сортировка по клику)
- **Приход** — добавить количество к существующему компоненту
- **Редактировать** — изменить количество, название или описание
- **Брак** — списать компоненты по браку с указанием причины
- **Новый компонент** — создать новый тип носителя или коробки
- **Удалить** — удалить тип компонента

**Двойной клик**:
- По колонке «Количество» — прямое редактирование числа
- По остальным колонкам — диалог редактирования

### Фиксация остатков

Позволяет сохранить снимок текущих остатков на текущую дату. Если в течение дня фиксация уже была, она перезаписывается (хранится только последняя).

### Просмотр фиксаций

Открывает окно, где можно:
1. Выбрать дату (ввод вручную или через календарь)
2. Нажать **«Показать»** — отобразится фиксация за эту дату (или ближайшую более раннюю)
3. Нажать **«Выгрузить в Excel»** — сохранить данные в файл

## Вкладка «Списание»

Пакетное списание инсталляционных комплектов.

1. В таблице отображаются все комплекты с колонками:
   - **Продукт** — название
   - **Описание**
   - **Доступно** — доступное количество
   - **Списать** — количество к списанию (по умолчанию 0)
2. Дважды кликните по ячейке **«Списать»** и введите количество
3. Нажмите **«Списать выбранное»** (красная кнопка справа)
4. Списание производится одной операцией с указанной датой отгрузки

**Кнопка «Очистить»** — сбрасывает все значения «Списать» в 0.

Строки с ненулевым количеством подсвечиваются жёлтым.

## Вкладка «История операций»

Журнал всех операций с возможностью фильтрации и экспорта.

### Фильтры

- **С / По** — фильтр по дате (два календаря)
- **Тип операции** — выпадающий список:
  - Добавлены носители
  - Добавлены коробки
  - Корректировка остатка
  - Списание комплекта
  - Списание по браку
- **Продукт** — фильтр по комплекту
- **Компонент** — фильтр по носителю или коробке

**«Показать»** — применить фильтры.
**«Сбросить»** — сбросить все фильтры.

### Таблица операций

| Колонка | Описание |
|---------|----------|
| Дата и время | Когда произошла операция |
| Тип операции | Вид операции |
| Продукт | Комплект или компонент |
| Количество | Сколько |
| Детали | Подробности: *DISC: Диск-1 (10 → 7)* |

**Сортировка** — по клику на заголовок столбца.

### Экспорт в Excel

Нажмите **«Экспорт в Excel»** — выберите имя и место сохранения. Формат: заголовки на синем фоне, границы, чередование цвета строк.

## Вкладка «Настройки»

### База данных

- В поле **«База данных»** отображается текущий путь к файлу БД
- Нажмите **«Обзор»** чтобы выбрать другой файл (поддерживаются локальные и сетевые UNC-пути)
- Нажмите **«Сохранить»** — все вкладки обновятся с данными из новой БД

### Версия

Отображается текущая версия приложения.

## Диалоговые окна

| Окно | Назначение | Клавиши / Особенности |
|------|------------|----------------------|
| Добавление комплекта | Создание нового ИК с составом | — |
| Редактирование комплекта | Изменение состава ИК | — |
| Приход | Добавление носителей/коробок на склад | Предзаполнение из выбранной строки |
| Корректировка остатка | Изменение количества, названия, описания | Спиннер для количества |
| Списание по браку | Списание бракованных компонентов | Поле «Причина» обязательное |
| Новый компонент | Создание нового типа носителя/коробки | — |

Все диалоги запоминают своё положение и размер.

### Элементы управления

- **CTkSpinbox** — поле с кнопками ▲/▼, используется для ввода количества (минимум 0)
- **DatePicker** — поле с кнопкой 📅, открывающей календарь для выбора даты

## Экспорт отчётов

### Отчёт по остаткам

На вкладке «Остатки» нажмите **«Выгрузить остатки в Excel»**.

Файл содержит три раздела:
- **ПРОДУКТЫ** — название + доступное количество комплектов
- **НОСИТЕЛИ** — название + остаток на складе
- **КОРОБКИ** — название + остаток на складе

### Фиксация остатков

Из окна «Просмотр остатков» нажмите **«Выгрузить в Excel»**.

### История операций

На вкладке «Операции» нажмите **«Экспорт в Excel»**.

## Сборка исполняемого файла

### Windows (.exe)

```powershell
.venv\Scripts\Activate.ps1
pyinstaller InstallKits.spec
```

Результат: `dist/InstallKits.exe`

Файл `settings.json` должен находиться рядом с `InstallKits.exe`.

### RED OS (Linux)

Для сборки под RED OS используйте папку `InstallKits-redos/`:

```bash
cd InstallKits-redos
./build_redos.sh
```

Подробнее — в `InstallKits-redos/README-REDOS.md`.
