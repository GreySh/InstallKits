# Учет инсталляционных комплектов

Программа для учета инсталляционных комплектов, состоящих из дисков и коробок.

## Возможности

- **Добавление новых комплектов** - создание новых видов продуктов с указанием состава (диски и коробки)
- **Управление остатками** - добавление новых дисков/коробок, корректировка количества
- **Списание комплектов** - списание комплектов с автоматическим уменьшением остатков компонентов
- **Просмотр остатков** - просмотр доступного количества комплектов по каждому продукту, а также остатков дисков и коробок
- **Отчеты** - формирование и экспорт отчетов в Excel (.xlsx)

## Установка и запуск

```bash
# Установить зависимости
poetry install

# Запустить приложение
poetry run python main.py
```

## Структура проекта

```
InstallKits/
├── main.py                    # Точка входа
├── config.py                  # Конфигурация
├── data/
│   ├── __init__.py
│   ├── db.py                  # Работа с базой данных
│   ├── products.py            # Управление продуктами
│   ├── discs.py               # Управление дисками
│   ├── boxes.py               # Управление коробками
│   ├── kits.py                # Управление составом комплектов
│   ├── stock.py               # Управление остатками
│   └── operations.py          # История операций
└── ui/
    ├── __init__.py
    ├── main_window.py         # Главное окно
    ├── tabs/
    │   ├── products_tab.py    # Вкладка продуктов
    │   ├── stock_tab.py       # Вкладка остатков
    │   └── view_stock_tab.py  # Вкладка просмотра
    ├── dialogs/
    │   ├── add_product.py     # Добавление продукта
    │   ├── add_disc.py        # Добавление диска
    │   ├── add_box.py         # Добавление коробки
    │   ├── add_all.py         # Добавление дисков/коробок
    │   ├── adjust_stock.py    # Корректировка остатков
    │   └── dispatch.py        # Списание комплекта
    └── report_window.py       # Окно отчетов
```

## Зависимости

- `customtkinter` - современный GUI
- `tinydb` - база данных
- `openpyxl` - экспорт в Excel
